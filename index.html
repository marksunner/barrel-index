Producer & model

Live rates come from OpenRouter’s open model catalogue and reflect published pay‑as‑you‑go API list prices. Subscription rates and Anthropic extra‑credit rates are not published anywhere as per‑token figures, so the observed entry is an estimate under your control — or, better, a measured rate computed from your own top-ups if a barrel-ledger.json file sits beside this page, or entered directly via each observed row’s Calibrate button (saved in this browser). To change which producers appear, set a permanent flagship model, or adjust the observed multiplier, edit the CONFIG block at the top of this file’s script. Dropdown changes last for the session; CONFIG changes are permanent. Auto‑refreshes every thirty minutes.
